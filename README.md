## Installation

- on Linux (terminal):

 > sudo apt install iverilog
 
- on Windows:

http://bleyer.org/icarus/
 
- on MacOS:

http://macappstore.org/icarus-verilog/

## Run and Compile Code

> iverilog -o output_name.out tb_name.v


> ./output_name.out
